# naughts and crosses
Play naughts and crosses on the command-line!

# How to play

First have python installed.

Linux:
```sudo apt update```
```sudo apt install python3```

Then:
```python3 game.py```

Enter a number 1-9 and then x or o.
e.g - 1x, 5o, 9x, etc.

Type in "r" to restart and "q" to quit.
